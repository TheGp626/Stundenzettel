# Stundenzettel PWA

digitaler Stundennachweis für Lübbert Event Interiors. gebaut weil ich keine Lust mehr hatte papier rumzuschleppen.

läuft als PWA, also installierbar auf android, funktioniert offline, sieht aus wie eine app.

## was es kann

- woche auswählen, tage aufklappen, zeiten eintragen
- mehrere schichten pro tag (z.b. auf- und abbau am selben tag)
- pause wird automatisch berechnet (0 / 30 / 45 min je nach stunden)
- minimum 3h pro schicht wird automatisch angerechnet
- AL name + unterschrift direkt im feld zeichnen (inklusive vollbild querformat modus)
- als PDF exportieren oder **direkt per Klick an die Buchhaltung per E-Mail** schicken
- wochen lokal speichern, wieder laden und verwalten
- **Monatsübersicht:** wochen werden automatisch nach monat (oder wählbarem stichtag) gruppiert. mit einem klick ein gesammeltes monats-pdf ausspucken.
- **Stundenlohn & Verdienst:** in den einstellungen den eigenen stundenlohn eintragen, dann zeigt er beim eintragen und im verlauf direkt an, was am ende pi mal daumen rauskommt (komplett privat im localstorage, wird *nicht* aufs PDF gedruckt).
- dark/light/auto theme (in den einstellungen).

## benutzen

1. ⚙️ einstellungen checken (stundenlohn, stichtag abrechnung)
2. name rein, woche auswählen, abteilung checken
3. tag aufklappen
4. von/bis eintragen, ort wenn nötig
5. AL unterschreiben lassen
6. speichern, monat vollmachen und dann direkt per knopfdruck an die buchhaltung schicken
